# 🍲 Hotpot – Food & Kitchen E-Commerce System

Hotpot is a modern interactive web-based food ordering system where users can customize meals, select ingredients, and rent or purchase kitchen equipment like burners and pots.

This project is built using **HTML, CSS, and Vanilla JavaScript**.

---

## 🚀 Features

### 🥩 Custom Food Builder
Users can create their own bowl by selecting:

- Meat Options: Chicken, Beef, Fish, Prawn  
- Vegetables: Lettuce, Radish, Egg, Tofu, Mixed Veggies  
- Sauces: Spicy, Garlic, Soy, Sweet Chili  

---

### 🍳 Kitchen Equipment System
- Rent or Buy Burners  
- Rent or Buy Pots  
- Quantity selection available  

---

### 🛒 Cart System
- Add/remove items dynamically  
- Live subtotal, delivery, and total calculation  
- Quantity update support  

---

### 📲 WhatsApp Order System
- Auto-generated WhatsApp message on checkout  
- Includes customer details + full order summary  

Example:

Customer Name: John Doe  
Phone: 0300XXXXXXX  
Address: Karachi  

Order Details:  
- Chicken Bowl = Rs. 1200  
- Burner Rent = Rs. 500  

Total: Rs. 1700
